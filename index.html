Modernised Build Engine

My first and most ambitious project is a game engine that is a modernized version of the old build engine.

The currently planned features are:

  • Viewport to view the game world
  • A default starting plane to build geometry on
    • Left click to create points on the default plane
    • Create edges from points
    • Create shapes from edges
    • Extrude shapes

About me

Just some guy who works with computers. But seriously I'm more or less a general programmer that has branched out from the world of C# and C++.
I like writing my own libraries and tools, going as far as to write my own UI rendering API in OpenGL. I also like having an intricate understanding of how something works. The more in-depth knowledge I have of how something works the better. This is why I love diving into technical manuals and reading documentation.

I have the most experience working with graphics libraries and prefer to work on graphics-related systems. However, I also have a bit of experience writing gameplay code.

Everything you see on this website is purely written by me. No APIs or frameworks, just pure HTML, CSS, PHP, and Javascript.

I have experience with the following:

  • C++
  • Unity Game Engine
  • HTML
  • CSS
  • FreeBSD
  • CLI (Command Line Interface)
  • OpenGL
  • Krita (Digital painting app)

Interests and hobbies include:
