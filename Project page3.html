Parry King

Project Details

A DDR-style game where you must parry the enemies attacks by swinging in the opposite direction. The game was made over the course of 3 days for an unofficial game jam.

Responsibilities

Programming the player's input & response to the enemy's attack, as well as managing version control for the project.
